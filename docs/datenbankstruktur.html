Datenbank-Schema

Übersicht über die PostgreSQL-Tabellenstrukturen und deren Relationen innerhalb des STN-CMS.

Sales (Verkäufe)

Speichert alle erfolgreich abgeschlossenen Transaktionen der Kunden.

  • ID PK UUID
  • Customer_Email String
  • Total_Amount Decimal
  • Sale_Date DateTime

Messages (Nachrichten)

Verwaltet Kundenanfragen und Kontaktformulardaten.

  • Msg_ID PK Int
  • Sender_Name String
  • Content Text
  • Is_Read Boolean

Products (Produkte)

Zentrale Tabelle für alle im Frontend angezeigten Artikel.

  • Prod_ID PK Int
  • Title String
  • Image_URL String
  • Stock_Quantity Int
