Voraussetzungen

Um das Projekt lokal auszuführen, müssen folgende Tools installiert sein:

  • git
  • React (v18+) für das Frontend.
  • .NET SDK (v7/v8) für das Backend.
  • PostgreSQL (oder Docker) für die Datenbank.

Download sources

Clone the project, either

via SSH:

git clone git@github.com:lemael/projectExplorer.git
cd projectExplorer

or HTTPS:

git clone https://github.com/lemael/projectExplorer.git
cd projectExplorer

Installieren Sie dependencies und local package

Bevor du das Projekt ausführst, müssen die Abhängigkeiten installiert werden:

Terminal / Bash 

# Frontend Abhängigkeiten
npm install

# Backend Abhängigkeiten
dotnet restore
