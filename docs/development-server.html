Development Server

1. Datenbank Verbindung

Bevor du den Server startest, stelle sicher, dass die PostgreSQL Verbindung in der Konfigurationsdatei korrekt hinterlegt ist:

Datei: backend/appsettings.Development.json

2. Backend starten (.NET)

Navigiere in den Backend-Ordner und führe den Befehl aus:

# In den Ordner wechseln
cd backend

# Server starten
dotnet run

API erreichbar unter: http://localhost:5000

3. Frontend starten (React TS)

Navigiere in den Frontend-Ordner und starte die Web-App:

# In den Ordner wechseln
cd frontend

# App starten
npm start

Dashboard erreichbar unter: http://localhost:3000
